How I do

My process is not unique but in only three steps can make magic happen.

Analysis

Once I receive the requirements of a customer or have a new idea, my process of analysis begin. I tend to make all possible questions to make sure I meet the user needs. In this stage I creat use cases, personas and my roadmap.

Design

The following stage start in paper. Through wireframes i make the first decisions about the design. Then, I make stylized designs with specialized programs and choose colors, fonts, and animations.

Code

Finally, I make my designs a reality using the latest web technologies and front-end development tools. I make sure that the website is fully functional in different browsers and screen resolutions to get the best results.
